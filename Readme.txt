Курсовая работа по базам данных
Need for Speed Underground 2

Описание проекта
Данный проект представляет собой базу данных компьютерной игры в жанре аркадных автогонок - Need for Speed Underground 2, разработанной в 2004 году. 

Таблицы, описание и столбцы:
Типы гонок
В Need for Speed: Underground 2 присутствует восемь видов гонок со своими правилами и особенностями.
•	Уникальный идентификатор.
•	Тип гонки.
Карьера
Основным режимом игры является «Карьера», в котором игрок должен побеждать в состязаниях, продвигаясь по сюжету. Всего в карьере 17 уровней по 5 гонок в каждом.
•	Уникальный идентификатор.
•	Гонка (уровень).
•	Этап уровня.
•	Тип гонки.
•	Денежное вознаграждение.
Профиль пользователя
•	Уникальный идентификатор.
•	Имя пользователя.
•	Количество пройденных гонок.
•	Количество денежных средств.
Автомобили
В режиме карьеры новые автомобили открываются по мере прохождения игры в магазинах в разных районах города. Автомобили в игре бесплатные.
•	Уникальный идентификатор.
•	Марка автомобиля.
•	Модель автомобиля.
•	Год выпуска.
•	Как получить автомобиль.
•	Что необходимо сделать для открытия автомобиля.
•	Необходимое количество пройденных гонок для открытия автомобиля.
•	Привод.
•	Рейтинг ускорения.
•	Рейтинг максимальной скорости.
•	Рейтинг управления.
•	Разгон от 0 до 60 миль в час.
•	Максимальная скорость (миль в час).
•	Максимальная скорость (км в час).
Магазины
В режиме «Карьера» новые автомобили можно купить в магазинах, которые открываются в разных районах города по мере прохождения игры, а также по городу можно находить магазины тюнинга автомобилей и гаражи.
•	Уникальный идентификатор.
•	Название магазина.
•	Цвет магазина.
•	Количество магазинов.
Улучшения
В игре присутствует большой выбор улучшений для автомобиля в визуальной части. Установить улучшения можно в соответствующем магазине.
•	Уникальный идентификатор.
•	Название улучшения.
•	Тип улучшения.
•	Стоимость улучшения.
Технические улучшения
У каждого автомобиля можно улучшить техническую составляющую. Установить улучшения можно в соответствующем магазине.
•	Уникальный идентификатор.
•	Название улучшения.
•	Тип улучшения.
Уровни улучшения технической части
У каждого параметра есть уровень (сток, уличный, профессиональный, экстримальный). По мере прохождения игры уровень детали можно повысить.
•	Уникальный идентификатор.
•	Название уровня улучшения.
Цены улучшений технической части

Чит-коды
Чит-коды можно вводить в главном меню игры для открытия бонусных автомобилей

Пользователи могут:
•	Создавать профиль.
•	Продвигаться по карьерной лестнице.
•	Участвовать в различных типах гонок.
•	Приобретать автомобили.
•	Устанавливать визуальные и технические улучшения на имеющиеся автомобили.
•	Вводить чит-коды для открытия бонусных автомобилей.
Триггер
•	Максимальное количество автомобилей в гараже – 5
Процедура
•	Добавление в таблицу «Купленные технические улучшения» всех деталей при покупке нового автомобиля по умолчанию. 
Представления
•	5 лучших машин, согласно их суммарному рейтингу.
•	Количество типов гонок во всей игре.
•	Стоимость максимального улучшения автомобиля.
•	Все установленные улучшения на автомобилях и их тип.
